# Arbeitsblatt 3: Eigenes Modell trainieren vs. Vorkonfigurierte Modelle nutzen

## Theorie-Teil: Modelltraining oder Fine-Tuning?
- API-Anbieter erlauben meistens kein direktes komplettes Training, sondern nur Fine-Tuning.
- Fine-Tuning bedeutet: bestehende Modelle an deine spezifischen Daten anpassen.
