# Arbeitsblatt 2: Vorteile, Nachteile und Limits API-basierter LLMs (Theorie & Analyse)

## Theorie-Teil: Vorteile & Nachteile API-basierter LLMs

### Vorteile:
- Keine lokale Infrastruktur nötig (kein GPU-Bedarf)
- Immer aktuelle Modelle
- Schneller Einstieg und einfache Integration in eigene Applikationen

### Nachteile:
- Kosten entstehen pro Anfrage
- Datenschutz und Privatsphäre: Daten gehen an externe Server
- Abhängigkeit von Anbieter, möglichen Downtimes oder API-Limits
